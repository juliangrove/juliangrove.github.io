Documentation

This module defines, from P, a datatype constructor D for representing terms with presuppositions within a dynamic semantics and defines from it an instance of the Effect class of Orchard, Petricek, and Mycroft. The result is an implementation of the framework for presupposition of chapter 3 of Grove 2019.

newtype D e a Source #

A datatype D for dynamic presupposition. D takes P and implicitly gives it ReaderT functionality so that it is sensitive to the discourse context, as well as (implicitly) StateT functionality so that it is sensitive to an incoming index.
