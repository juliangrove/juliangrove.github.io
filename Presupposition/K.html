Documentation

This module defines, from D, a datatype constructor K for representing terms with presuppositions dynamically and in a continuized format so that one can analyze the semantic contribution of indefinites. We define from K an instance of the PMonad class of Orchard Petricek, and Mycroft. Although K constitutes a parameterized monad in the implementation, these gives rise to graded monads (in real life), given a monoid over the parameters. The result is thus a faithful implementation of the framework for presupposition of chapter 3 of Grove 2019.
