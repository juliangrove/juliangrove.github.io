Are you a native American English speaker?

Do not complete this HIT unless you are a native American English speaker. We will reject submissions that fail to answer more than half of the questions correctly.

Instructions

This test is designed to confirm that you are a native speaker of American English. To be a native speaker, you must have learned American English before the age of 5, and not be more proficient in any other language besides English (of any sort). Correctly answering seven or more of the nine questions below will qualify you for future HITs that we post requiring native American English speakers.

Several of the questions will ask you to decide whether sentences are "acceptable." A sentence is acceptable if it sounds like something a native American English speaker would say. It doesn't have to be "proper" like you were taught in school; it just has to sound natural.

Questions

 

Which of the following sentences is acceptable? Select ALL that apply.

 

Which of the following continuations is acceptable? Select ALL that apply.

Ethan reads more books...

 

Which of the following questions is acceptable? Select ALL that apply.

 

Which of the following continuations is acceptable? Select ALL that apply.

I was talking to Susan about John. She said...

 

Which of the following sentences is acceptable? Select ALL that apply.

 

Which of the following continuations of the sentence is acceptable? Select ALL that apply.

If you walk barefoot on the beach,...

 

Which of the following continuations of the sentence is acceptable? Select ALL that apply.

I talked to Tom, and he said...

 

Which of the following would be an acceptable answer to the following question? Select ALL that apply.

Did Steven take out the trash this morning?

 

Fill in the blank:

The girl who lived in the house on the corner asked ___ father to mow the lawn.
